IoT Fall Detection System

Overview

This project is an IoT-enabled fall detection system designed to be embedded into wearable clothing for elderly users. The system detects falls using a combination of inertial sensing and proximity sensing, and then triggers alerts and remote notifications to caregivers or authorities.

Problem Statement

Falls among elderly individuals often go undetected, especially if the person is unconscious or unable to call for help. The goal of this system is to detect a fall reliably, minimize false positives, and immediately trigger both local and remote alerts.

System Design

The system combines multiple sensors to improve reliability:

  • MPU6050 accelerometer/gyroscope for tilt and orientation detection
  • Ultrasonic sensor for sudden proximity changes
  • DHT11 temperature sensor for environmental monitoring
  • GT-U7 GPS module for location tracking
  • ESP32 microcontroller for processing and IoT communication
  • NFC tag for storing medical information

Operation Logic

The system continuously reads sensor data and evaluates fall conditions based on:

  • Body tilt exceeding approximately 45 degrees
  • Sudden proximity change detected by ultrasonic sensor

When a fall is detected:

  • Temperature and GPS data are transmitted via IoT (Adafruit platform)
  • GPS location is sent via email for emergency response
  • Buzzer and vibration motor are activated to alert the user and surroundings

When the user returns to a stable upright position, local alerts stop while emergency data remains available for responders.

Key Contributions

  • Integrated 6 different sensors into a unified embedded system
  • Developed logic to reduce false positives using multi-sensor validation
  • Implemented IoT communication for real-time monitoring
  • Designed alert system using vibration and audio feedback

Challenges

One major challenge was handling inconsistent GPS data formatting and signal acquisition delays, which affected timing in the control loop. Additionally, integrating sensors originally designed for Arduino required significant effort in parsing and synchronization.

Outcome

The final system successfully demonstrated multi-sensor fall detection with reduced false positives compared to single-sensor systems.
