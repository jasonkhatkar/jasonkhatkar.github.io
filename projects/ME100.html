IoT Fall Detection System

Embedded multi-sensor wearable system designed to detect falls in real time and trigger local + IoT-based emergency alerts for elderly users.

Problem

Falls among elderly individuals often go undetected when the user is unconscious or unable to call for help. Existing single-sensor solutions suffer from high false positive rates or unreliable detection in real-world conditions.

The goal of this system was to design a low-cost, wearable embedded solution that improves detection reliability while enabling immediate emergency communication.

System Overview

Multi-sensor embedded system combining inertial sensing, proximity detection, environmental monitoring, and IoT communication using ESP32.
  • MPU6050 accelerometer/gyroscope for motion + orientation detection
  • Ultrasonic sensor for sudden proximity change detection
  • DHT11 sensor for environmental temperature monitoring
  • GT-U7 GPS module for real-time location tracking
  • ESP32 microcontroller for real-time processing + IoT communication
  • NFC tag for storing medical information

Detection Logic

Fall detection is triggered when multiple conditions are satisfied simultaneously:

  • Body tilt exceeds ~45° threshold (IMU-based detection)
  • Sudden proximity change detected via ultrasonic sensor

This multi-sensor validation approach reduces false positives compared to single-sensor systems.

System Response

  • ESP32 sends GPS + temperature data via Adafruit IoT platform
  • Email alert sent containing real-time location data
  • Buzzer + vibration motor activated for local alerting
  • System resets alerts when stable upright posture is restored

Key Contributions

  • Integrated 6 heterogeneous sensors into a unified embedded system
  • Designed multi-sensor fusion logic to improve reliability
  • Implemented IoT communication pipeline using ESP32
  • Developed real-time alert system with both local + remote response

Challenges

Major challenges included inconsistent GPS data formatting and latency in signal acquisition, which impacted real-time synchronization in the control loop.

Additional integration effort was required to synchronize sensor data streams across different communication protocols.

Outcome

Demonstrated reliable multi-sensor fall detection with significantly reduced false positives compared to single-sensor baseline approaches.
