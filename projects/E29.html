E29 Group 6 - Multi-Function Phone Case System

Overview

This project involved designing a multi-functional phone case system integrating protective casing with embedded utility tools. The focus was on balancing manufacturability, material selection, and functional fit between components.

Summary

The design consists of multiple layered components including an inner flexible case, rigid outer structure, and integrated tool set. Material selection was driven by tradeoffs between flexibility, durability, precision, and manufacturing constraints.

Key Focus Areas

  • Material selection and process compatibility
  • Fit and tolerance design between components
  • Manufacturability using available lab processes
  • Integration of functional tools into structural design
