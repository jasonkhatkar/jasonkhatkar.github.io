ME136 – Quadcopter Estimation & Control

MATLAB-based control systems project focused on quadcopter state estimation, sensor fusion, and hierarchical flight control design.

Overview

Developed a full simulation and control pipeline for a quadcopter system with the goal of improving hover stability and trajectory tracking. Focused on integrating sensor fusion, state estimation, and multi-layer control architecture.

My Contributions

  • Developed simulation and control logic for quadcopter dynamics in MATLAB
  • Implemented sensor fusion and state estimation using IMU and optical flow data
  • Designed and tuned hierarchical control loops (position → velocity → attitude)
  • Analyzed system performance and produced simulation results for reporting

System Architecture

  • IMU-based attitude estimation (gyro integration + accelerometer correction)
  • Optical flow–based velocity estimation with height correction
  • Position estimation via integrated velocity model
  • Closed-loop hierarchical control structure

Key Engineering Improvements

  • Introduced position control layer to improve hover stability
  • Tuned control gains to reduce overshoot and oscillations
  • Improved velocity estimation robustness under sensor noise
  • Enhanced overall stability through iterative parameter tuning

Results

Achieved approximately 72 seconds of stable simulated flight before instability due to accumulated sensor drift and optical flow noise.

Performance was primarily limited by estimator accuracy rather than controller design, highlighting the importance of sensor fusion robustness in real-world systems.

Key Limitations

  • Optical flow noise introduced drift in velocity estimation
  • Position error accumulated over long-duration simulation
  • System performance bounded by estimator rather than control design
