ME136 – Quadcopter Estimation and Control Lab

Overview

This project focused on the modeling, estimation, and closed-loop control of a quadcopter system. The goal was to improve hover stability and flight performance using sensor fusion, state estimation, and hierarchical control design.

My Contributions

  • Implemented state estimation and sensor fusion logic
  • Designed and tuned control loops for stability and performance
  • Created plots, analysis, and conceptual explanations for report sections

System Description

The quadcopter system was modeled using IMU (accelerometer + gyroscope), optical flow, and height sensor data. A complementary estimator was used to compute roll, pitch, yaw, position, and velocity states.

Control Architecture

  • Attitude estimation using gyro integration + accelerometer correction
  • Velocity estimation using optical flow and height correction
  • Position estimation via velocity integration
  • Hierarchical control: position → velocity → attitude → motor commands

Key Improvements Made

  • Added position control loop to improve hover stability
  • Tuned rate and angle time constants for damping performance
  • Improved horizontal velocity estimation stability
  • Adjusted control gains to extend stable flight duration

Results

The system achieved approximately 72 seconds of stable flight before instability and wall collision. The main limiting factors were sensor noise (especially optical flow) and positional drift over time.

Limitations

  • Optical flow noise introduced instability in horizontal velocity estimation
  • Position drift accumulated due to sensor uncertainty
  • Performance limited by estimator accuracy rather than controller structure
