Automated Pill Dispenser (ME102B)

Mechatronics Mechanical Design Embedded Systems

Overview

Designed and built an automated pill dispensing system to improve medication adherence and support users with cognitive or physical challenges. The system integrates mechanical actuation, control logic, and embedded hardware into a single coordinated platform.

Problem

Many individuals struggle with managing medication schedules due to age-related memory decline or disability. This project aimed to create a reliable mechanical solution to automate and simplify pill delivery.

Design Approach

  • Rack and pinion-based linear motion system for controlled caddy positioning
  • Single-motor architecture for simplicity and reliability
  • Modified front plate design with bushings for load support
  • Use of constrained guide rails to ensure consistent motion

Key Engineering Decisions

  • Added bushings to support cantilevered shaft loads
  • Used laminated front plate with wooden dowels and 4-40 fasteners
  • Switched from free-floating rack to constrained aluminum rail system
  • Selected single MCU design to avoid inter-controller communication complexity

System Notes

The system was tuned empirically due to lack of encoders, relying on PWM adjustment and iterative testing. While this reduced precision, it simplified system integration and improved reliability within project constraints.

Results

The final system successfully achieved functional automated dispensing with stable mechanical motion and repeatable positioning.
